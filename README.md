# SKS Trainer App
For anybody in Germany who wants to learn the theory of the SKS sailing certificate - here is an app which helps you learn the ~600 theory questions.

## Main files

- `extract_sks_questions_v5.py`: Extracts questions and answers from the `Fragenkatalog-SKS.pdf` into structured JSON (and optionally extracts embedded images).
- `sks.json`: The resulting JSON question bank used by apps and the verification webpage.
- `index.html`: A simple verification webpage that loads `sks.json` and displays each question, answer, and any extracted images for easy comparison with the PDF.
- `Fragenkatalog-SKS.pdf`: The source PDF containing the official exam questions (not generated by this repo).
- `pictures/`: Output folder where extracted images are stored when running the extractor.

## Usage
1. Run the extractor: `python extract_sks_questions_v5.py`
2. Open the verification page:`http://localhost:8000/index.html` (after running `python -m http.server 8000` in this repo).
